# MIPS VM
A virtual machine emulating a MIPS virtual machine.

## Project structure
- `mips-vm.rkt` is the root project
- `constants.rkt` stores only constants (e.g. bitmasks, opcodes, etc.)

## Useful notes
`racket -l errortrace -t mips-vm.rkt`
